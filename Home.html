Welcome to Script This!

Script This is your go-to web-based tool for managing Lua scripts and visualizing JSON data, specifically designed for GTA 5 mod menus like Cherax and Kiddions. Whether you are a beginner or a seasoned script developer, Script This provides intuitive tools and features that streamline your workflow.

Features

  • Manage and edit Lua scripts with ease.
  • Visualize and manipulate JSON data with our interactive viewer.
  • Switch between code view and tree view for a better data understanding.
  • Customizable settings to suit your scripting needs.
